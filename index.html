Java Projects

Android Studio Mobile Attendance Application

4th year Project overview:

For my 4th year project in college, I created a mobile application using Java which allows students to scan their student card and sign them into classes. This application was an idea to replace the current paper system that my college had been using for decades. A student would scan their student card and based on the current date and time, the student would be signed into that class without needing to manually mark themselves as present.

What I Learned:

  • Reading Files from Android File System and structuring them into variables
  • NFC: How to successfully authenticate and read sectors

Work Placement

SAP Ireland (7 Months)

During my internship at SAP, I joined a team where I was developing Web Applications for use by the SAP Teams across the world. One of these applications was a JavaScript - Diary Management Website which would allow managers to assign tasks to their employees. Every employee can see only the tasks allocated to them and have the option to mark the tasks as complete and allow them to message their manager through the application. Managers have the ability to monitor tasks for all of their employees.

Participating in and completing this project helped me understand how my work in college relates to the working world. I was finally able to understand and visualize the lifecycle of a project as I was actively involved with a customer for feedback and external opinions.
